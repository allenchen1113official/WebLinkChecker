WebLinkChecker

開源網站失效連結檢查工具
自動爬取全站連結，即時回報失效網址與來源頁面

查看原始碼 快速安裝

功能特色

無需登入，免費開源，本機執行保護隱私

🕷️

全站爬取

從起始網址出發，自動遞迴爬取同網域下所有頁面，發現每一個 <a href> 連結。

🔍

即時串流結果

透過 Server-Sent Events 即時推送每條連結的檢查結果，無需等候全部完成。

📍

標示來源頁面

每個失效連結都會顯示是在哪些頁面被找到的，方便直接定位並修復。

🌐

網頁操作介面

內建 Flask 網頁應用程式，提供視覺化操作介面，支援篩選、匯出 CSV。

⌨️

命令列工具

同時提供 CLI 工具，可整合至 CI/CD 流程，發現失效連結時回傳 exit code 1。

📤

CSV / JSON 匯出

支援將完整報告匯出為 CSV 或 JSON 格式，方便後續分析或存檔。

安裝與使用

需要 Python 3.10+

1. 下載並安裝套件

# git clone https://github.com/allenchen1113official/WebLinkChecker.git
$ cd WebLinkChecker
$ pip install -r requirements.txt

2a. 啟動網頁界面

$ python app.py
# 開啟瀏覽器前往 http://localhost:5000

2b. 使用命令列工具

$ python link_checker.py https://example.com
$ python link_checker.py https://example.com --output report.csv
$ python link_checker.py https://example.com --max-pages 50 --verbose

常用選項

CLI 選項
--timeout SEC請求逾時秒數（預設 15）
--delay SEC請求間隔秒數（預設 0）
--max-pages N最大爬取頁數（0=不限）
--output FILE儲存報告（.csv 或 .json）
--broken-only匯出時只含失效連結
--verbose / -v顯示每個 URL 的檢查過程
Exit Code
0  無失效連結
1  發現一個或多個失效連結
可整合至 CI/CD，在有失效連結時讓 pipeline 失敗。
